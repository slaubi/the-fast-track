Me dê o livro do Symfony 5!

É possível construir rapidamente uma aplicação web moderna e amar o processo? Com o Symfony 5, você pode. Este livro - escrito pelo criador do Symfony - apresenta uma abordagem pragmática para desenvolver aplicações web com o Symfony 5: do zero à produção. Se você está descobrindo o Symfony pela primeira vez ou refrescando seu conhecimento, este guia prático fornece a introdução definitiva às aplicações modernas usando o Symfony.

Saiba como combinar o Symfony com Docker, APIs, notificações, filas e tarefas assíncronas, Webpack, Single-Page Applications e outras ferramentas poderosas. Ao final deste livro, você terá a confiança, compreensão e um caminho claro para mergulhar e começar a construir sua próxima aplicação com o Symfony.
