Dammi il libro di Symfony 5!

È possibile costruire rapidamente una moderna applicazione web  
e amarne il processo? Con Symfony 5 è possibile. Questo libro, scritto dal creatore di Symfony, delinea un approccio pragmatico allo sviluppo di applicazioni web con Symfony 5: da zero alla produzione. Sia che si scopra Symfony per la prima volta o che si rinfreschino le proprie conoscenze, questa guida pratica fornisce l'introduzione definitiva alle moderne applicazioni Symfony.

Imparate a combinare Symfony con Docker, API, notifiche, code e task asincroni, Webpack, applicazioni SPA (Single Page Application) e altri strumenti potenti. Alla fine di questo libro, avrete la fiducia, la comprensione e un chiaro percorso per tuffarvi in Symfony e iniziare a costruire la vostra prossima applicazione.
