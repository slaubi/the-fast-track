¿Cuál es su último deseo, Amo Programador?

¡Dame el libro de Symfony 5!

¿Es posible construir rápidamente una aplicación web moderna y disfrutar durante el desarrollo? Con Symfony 5, puedes lograrlo. Este libro - escrito por el creador de Symfony - presenta un enfoque pragmático para desarrollar aplicaciones web con Symfony 5: desde cero hasta producción. Ya sea que estés descubriendo Symfony por primera vez o refrescando tus conocimientos, esta práctica guía proporciona una introducción definitiva a las aplicaciones Symfony modernas.

Aprende a combinar Symfony con Docker, APIs, notificaciones, colas y tareas asíncronas, Webpack, aplicaciones de una sola página (SPA) y otras potentes herramientas. Al final de este libro, tendrás la confianza, la comprensión y un camino claro para sumergirte y empezar a construir tu próxima aplicación con Symfony.
